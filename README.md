# Проект: **Проектная работа "Mesto" 

## Описание проекта:
Адаптивный одностраничный сайт-портфолио, посвящённый путешествиям по России, разработанный на основе макета из Figma.
Интерфейс корректно отображается на устройствах с разным разрешением экрана — от 320 до 1280 пикселей и выше.

## Технологии:
JavaScript
CSS Grid
Flexbox
Медиазапросы
Подключение пользовательских шрифтов
Методология БЭМ (Nested)

## Инструкция:
Открыть файл index.html
